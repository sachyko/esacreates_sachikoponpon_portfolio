About the Designer

Sachiko Ponpon

ポンポン サチコ

Webデザインへの道は、好奇心から始まり、実践を重ねる中で深まっていきました。  トレーニングの一環として複数のWeb開発プロジェクトを経験し、 現在は「分かりやすさ」「使いやすさ」「見た目の美しさ」のバランスを大切にした UIデザインに取り組んでいます。

HTML、CSS、JavaScriptの実装経験があり、過去にReactを用いた制作にも取り組みました。 現在は、アイデアをレスポンシブなWebサイトへと落とし込むことを意識し、 レイアウト調整や情報設計、直感的なUIデザインを中心に取り組んでいます。

東京を拠点に、ユーザー目線と細部へのこだわりを大切にしながら制作しています。デザイン以外の時間には、街を歩いたり、静かなカフェで過ごしたりする中で、色やリズム、空気感から日々インスピレーションを得ています。

Design Skills & Tools

Webサイトデザイン / バナー制作

Figma

Figmaを中心に、WebサイトやLP、バナーの
デザイン制作を行っています。レイアウトの分かりやすさや余白のバランスを大切にし、直感的に伝わるデザインを心がけています。

Photoshop

Photoshopを使用して、Webデザインに必要な画像加工やビジュアル調整を行っています。写真のトーン補正やトリミング、パーツ制作など、デザイン全体の統一感を大切にしています。

コーディング

HTML

HTMLを用いて、情報構造を 整理し、コンテンツの意味が 正しく伝わるマークアップを 行っています。見出しや階層を意識し、分かりやすい 構造設計を心がけています。

CSS

CSSを用いて、レイアウトや余白、配色を調整し、デザインを視覚的に整えています。レスポンシブ対応を意識し、画面サイズに応じた見やすい 表現を大切にしています。

JavaScript

Webサイトにインタラクティブな動きや動的な表現を加え、スムーズな操作性とレスポンシブな機能によってユーザビリティの向上を
図っています。

React

過去にReactを用いた制作経験があり、コンポーネント構造や基本的な考え方を理解しています。 現在は デザインを主軸に、実装とのつながりを意識したUI設計に
活かしています。
